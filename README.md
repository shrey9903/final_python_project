# Stock Market Dataset Analysis

**Name:** Shrey Patel\

## Project Overview

This project analyzes a stock market listing dataset obtained from
Kaggle. The dataset contains information about securities traded across
different stock exchanges such as NASDAQ, NYSE, and NYSE American. The
analysis focuses on understanding the distribution of securities,
identifying market categories, ETF classifications, and examining
trading characteristics.

## Objectives

-   Explore the structure of the stock market dataset
-   Perform descriptive statistical analysis
-   Identify patterns in securities listed across exchanges
-   Visualize distributions and relationships between dataset variables
-   Demonstrate data analysis techniques using Python

## Tools and Libraries Used

-   Python
-   Pandas (data manipulation)
-   NumPy (numerical analysis)
-   Matplotlib (data visualization)
-   Seaborn (advanced visualization)
-   KaggleHub (dataset download)

## Dataset Description

The dataset contains information about securities listed on major U.S.
stock exchanges.

### Key Columns

-   **Nasdaq Traded:** Indicates whether the security is traded on
    NASDAQ (Y/N)
-   **Symbol:** Ticker symbol of the security
-   **Security Name:** Full name of the company or financial instrument
-   **Listing Exchange:** Exchange where the security is listed
-   **Market Category:** Classification of the market segment
-   **ETF:** Indicates whether the security is an Exchange Traded Fund
-   **Round Lot Size:** Standard number of shares typically traded
-   **Test Issue:** Indicates if the symbol is used for testing
-   **Financial Status:** Financial condition of the company
-   **CQS Symbol:** Consolidated quotation system symbol
-   **NASDAQ Symbol:** NASDAQ specific ticker
-   **NextShares:** Indicates if the security belongs to the NextShares
    category

## Analysis Performed

The following analyses were conducted:

1.  Dataset exploration and inspection
2.  Checking missing values
3.  Summary statistics of numerical variables
4.  Grouping securities by exchange and market category
5.  Distribution analysis of ETF and non‑ETF securities
6.  Visualization of market distribution and trading characteristics

## Visualizations Included

The project includes several graphical analyses:

-   Bar charts (exchange and category distributions)
-   Scatter plots
-   Line graphs
-   Histograms
-   Pie charts
-   Box plots
-   Correlation heatmap

These visualizations help identify patterns in stock listings and
trading characteristics.

## Key Insights

-   Most securities are listed on major exchanges such as NASDAQ.
-   The majority of securities are standard company stocks rather than
    ETFs.
-   Round lot sizes are commonly standardized across securities.
-   Financial status values indicate that most listed securities are in
    normal financial condition.

## Conclusion

This project demonstrates how Python-based data analysis tools can be
used to explore financial datasets. Through statistical analysis and
visualization, we gain insights into how securities are distributed
across exchanges and categories within the stock market.

The techniques used in this project can be extended for deeper financial
analytics, stock trend analysis, and predictive modeling.
